# FlexGrow

FlexGrow is a simple wrapper to add the `flex-grow` or `flex-shrink` css [Flexbox API](https://www.w3schools.com/css/css3_flexbox.asp) properties.

To use this component it needs to be a child of the [Flex](/story/base-flex-intro--page) or a child of any flex element
