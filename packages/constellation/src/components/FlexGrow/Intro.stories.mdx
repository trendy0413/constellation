# FlexGrow

FlexGrow is a simple wrapped to add the `flex-grow` css [Flexbox API](https://www.w3schools.com/css/css3_flexbox.asp) property.

To use this component it needs to be a child of the [Flex](/story/components-flex-intro--page) or a child of any flex element
